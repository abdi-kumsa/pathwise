PHASE 03
Machine Learning Model Development
Train three separate models, evaluate them, and save them as files the API will load for inference.
Weeks 3 – 5
Abdi
All three ML models + SHAP explainability
Train the XGBoost placement prediction model
Prompt Claude Code: "Write /ml/train_placement_model.py. Load features.parquet. Target variable: placed (1/0). Features: gpa, internship_count, project_count, communication_score, program (encoded). Split 80/20 train/test. Train XGBoostClassifier. Print classification report and ROC-AUC score. Save model to /ml/models/placement_model.pkl using joblib."
ML
Add SHAP explainability to the placement model
Prompt Claude Code: "Extend train_placement_model.py to: after training, compute SHAP values using shap.TreeExplainer on the test set. Save mean absolute SHAP values per feature as /ml/models/shap_values.json. This will be used by the frontend to show a feature importance chart."
ML
Train the Prophet enrollment forecasting model
Prompt Claude Code: "Write /ml/train_enrollment_forecast.py. Load enrollment data from Postgres grouped by program and year. For each program, train a separate Prophet model with yearly seasonality. Forecast 3 years forward. Save all models to /ml/models/prophet_models/ as one .pkl per program. Save forecast outputs as /ml/models/enrollment_forecasts.json."
ML
Train the LSTM model for temporal demand patterns
Prompt Claude Code: "Write /ml/train_lstm_demand.py. Use Keras. Load enrollment features.parquet with lag and rolling average columns. Build a 3-layer LSTM: LSTM(64) → Dropout(0.2) → LSTM(32) → Dense(1). Train for 50 epochs, batch size 16. Use MAE as loss. Print train vs validation MAE. Save model to /ml/models/lstm_demand.h5 and the scaler to /ml/models/lstm_scaler.pkl."
ML
Build a model evaluation report
Prompt Claude Code: "Write /ml/evaluate_models.py that loads all saved models, runs them on test data, and outputs a report: XGBoost ROC-AUC and F1, Prophet MAPE per program, LSTM MAE and RMSE. Save as /ml/models/evaluation_report.json." Review and confirm results are reasonable before proceeding.
ML
Build the salary prediction model
Prompt Claude Code: "Write /ml/train_salary_model.py. Use XGBoostRegressor. Features: program (encoded), gpa, internship_count, top skills from job_postings. Target: salary. Train, evaluate with RMSE. Save to /ml/models/salary_model.pkl."
